### Tools

- `otool` - display specified parts of object files or libraries
- `nm` - list symbols from object files
- `lipo` - create or operate on universal files
- `file` - determine file type
- `codesign` - create, check, and display code signatures
- `dwarfdump` - display the contents of DWARF debug sections
- `strings` - find and print strings in files
- `addr2line` - convert addresses into file names and line numbers
- `readelf` - display information about ELF files
